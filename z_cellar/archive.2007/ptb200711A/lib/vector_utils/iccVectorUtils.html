vector_utils

vector_utils funtions provide a set of routines for performing vector math functions. These functions follow a common vector calling protocol. The protocol followed usually takes the form:

vectorFunction1( FLT32 *pointerToSourceArray1, INT32 sourceArray1StepIndex,

FLT32 *pointerToSourceArray2, INT32 sourceArray2StepIndex,

FLT32 *pointerToDestinationArray, INT32 destinationArray StepIndex,

UNT32 numberOfElementsToProcess);


The functions for vector add, vector subtract, vector multiply, vector divide , all fit this form exactly. Other vector funtions, though, would just be a variation of this theme.
