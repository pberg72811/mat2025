The Boot Server page



To create a server that you can install Redhat LINUX from:



Set up an export directory for “NFS-ing” files:

NotesBootServerExport



Set up PXE:

NotesBootServerPXE



Set up a TFTP directory:

NotesBootServerTFTP



Set up DHCP:

NotesBootServerDHCP



Finish setting up PXE:

NotesBootServerPXEFinish



To install RedHat over network using the new installation server:

NotesInstallFromServer



Once Redhat is installed set up the STE environment:

NotesBootSTE



To Build the Kernel:

NotesBootKernelBuild



To configure the Kernel:

Notes09BootKernelConfig
