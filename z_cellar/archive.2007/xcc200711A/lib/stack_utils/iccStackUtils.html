stack_utils

stack_utils provides a set of functions for the creation, operation, and destruction of stacks.

A stack is the combination of an allocated segment of data memory plus an index counter. The data is treated as a "First In Last Out", FILO, buffer. The way to add data to the stack is to place values in the data memory one at a time and increment the index counter each time. This procedure is call "pushing". The way to retreive data is to read the data value at the point indicated by the index counter and then decrement the counter. This procedure is called "poping".
